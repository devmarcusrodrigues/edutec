FAQ, Bug Report e Sobre Nós

O site está todo quebrado, o que devo fazer?

Talvez seu JavaScript esteja desabilitado em seu navegador.
Recomendamos usar o Google Chrome e habilitar o uso de JavaScript via Três Pontinhos
> Configurações > Privacidade e Segurança > Configurações do Site > JavaScript > Selecionar “Os sites podem usar JavaScript”

  • Para o Microsoft Edge: https://support.microsoft.com/pt-br/microsoft-edge
  • Para o Mozilla Firefox: https://support.mozilla.org/pt-BR/home
  • Para o Apple Safari: https://www.apple.com/br/support/safari/

O site está estranho no celular. Isso é um bug?

Não, não é. Está assim justamente por restrições de tempo que tivemos durante um período bem turbulento, com prazos bem curtos junto de outros trabalhos escolares.
Pedimos perdão pelo inconveniente, e talvez em um update próximo possamos criar a responsividade do site.
